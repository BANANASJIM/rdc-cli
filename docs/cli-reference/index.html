    CLI Reference

    rdc

    rdc: Unix-friendly CLI for RenderDoc captures.

    Usage:

    rdc [OPTIONS] COMMAND [ARGS]...

    Options:

    Name Type Description Default
    --version boolean Show the version and exit. False
    --session text Named session (default: value of $RDC_SESSION or 'default'). None
    --help, -h boolean Show this message and exit. False

    assert-clean

    Assert capture log has no messages at or above given severity.

    Usage:

    rdc assert-clean [OPTIONS]

    Options:

    Name Type Description Default
    --min-severity choice (HIGH | MEDIUM | LOW | INFO) Minimum severity threshold. HIGH
    --json boolean JSON output. False
    --help, -h boolean Show this message and exit. False

    assert-count

    Assert a capture metric satisfies a numeric comparison.

    Usage:

    rdc assert-count [OPTIONS] {draws|events|resources|triangles|passes|dispatches
                 |clears|shaders}

    Options:

    Name Type Description Default
    --expect integer Expected count value. Sentinel.UNSET
    --op choice (eq | gt | lt | ge | le) Comparison operator. eq
    --pass text Filter by render pass name. None
    --json boolean JSON output. False
    --help, -h boolean Show this message and exit. False

    assert-image

    Compare two images pixel-by-pixel.

    Exit 0 if images match (within threshold), exit 1 if they differ, exit 2 on error (size mismatch, invalid image).

    Usage:

    rdc assert-image [OPTIONS] EXPECTED ACTUAL

    Options:

    Name Type Description Default
    --threshold float Diff ratio threshold (%). 0.0
    --diff-output file Write diff visualization PNG. None
    --json boolean JSON output. False
    --help, -h boolean Show this message and exit. False

    Execute application and capture a frame.

    Usage: rdc capture [OPTIONS] -- EXECUTABLE [APP_ARGS...]

    Usage:

    rdc capture [OPTIONS]

    Options:

    -o, --output path Output .rdc file path. Sentinel.UNSET

    --frame integer Queue capture at frame N. None
    --trigger boolean Inject only; do not auto-capture. False
    --timeout float Capture timeout in seconds. 60.0
    --wait-for-exit boolean Wait for process to exit. False
    --keep-alive boolean Keep target process running after capture. False
    --auto-open boolean Open capture after success. False
    --api-validation boolean Enable API validation. False
    --callstacks boolean Capture callstacks. False
    --hook-children boolean Hook child processes. False
    --ref-all-resources boolean Reference all resources. False
    --soft-memory-limit integer Soft memory limit (MB). None

    --help, -h boolean Show this message and exit. False
